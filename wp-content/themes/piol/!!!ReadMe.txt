Theme Name: Piol
Theme URI :http://c7sky.com/
Description: Designed by http://c7sky.com/
Author: Cople
Version: 1.0.1
Tags: personal, pix, white, two-columns, custom-header, custom-background, threaded-comments

特性：
・ 支持WordPress 3.0自定义菜单
・ 支持WordPress 3.0自定义背景
・ 自带3个侧边栏小工具
・ 自带后台主题设置面板
・ 兼容IE6~IE9、Firefox、Opera、Chrome(Webkit内核)等主流浏览器

缺陷：
・ 应该只支持WordPress 3.0+
・ 默认只支持4层嵌套回复的样式
・ 通过HTML5验证，没过CSS验证...因为有CSS3的属性和一些css hack
・ 使用DD_belatedPNG.js解决IE6的PNG图像透明问题
・ 导航菜单不支持二级列表

插件支持：
・ Simple Tags
・ WP-PageNavi
・ WP-PostViews
・ WP-Syntax
・ Clean Archives Reloaded
・ Mail To Commenter
